# threeJs-first-project :three:

### TO START

1. download three.js (https://threejs.org/)

- add the `three.js` file in the folder

- on the command line type:
`php -S 127.0.0.1:8080`

- on the browser go to
`http://127.0.0.1:8080`

:tada: :tada: :tada:
